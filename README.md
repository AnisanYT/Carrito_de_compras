
# Carrito de compras




## Run Locally

Clone the project

```bash
  git clone https://github.com/AnisanYT/Carrito_de_compras.git
```

Go to the project directory

```bash
  cd Carrito_de_compras
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
